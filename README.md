# Improved (IMHO) Key Bindings for the Windows Terminal

## Bindings

### Duplicate pane
#### Auto Horizontal (-) or Vertical (|)
`ALT + SHIFT + D` or `CTRL + D`

#### Horizontal (-)
`CTRL + SHIFT + Down Arrow` or `CTRL + SHIFT + Up Arrow`

#### Vertical (|)
`CTRL + SHIFT + Left Arrow` or `CTRL + SHIFT + Right Arrow`

### Move Focus
`CTRL + Arrow` 

### Close Pane
`CTRL + F4`
